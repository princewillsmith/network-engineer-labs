# Linux Commands

## Check IP Address

```bash
ip addr
```

Displays all network interfaces and IP addresses.

---

## Show Routing Table

```bash
ip route
```

Shows the routing table.

---

## Check Disk Usage

```bash
df -h
```

Displays disk usage in a human-readable format.

---

## Check Memory Usage

```bash
free -m
```

Shows RAM usage.

---

## List Files

```bash
ls -la
```

Lists all files, including hidden files.

---

## Current Directory

```bash
pwd
```

Displays the current working directory.

---

## Search for a File

```bash
find / -name filename
```

Searches for a file by name.

---

## Check Listening Ports

```bash
ss -tulnp
```

Displays listening TCP and UDP ports.

---

## Check Running Processes

```bash
ps -ef
```

Lists all running processes.

---

## Ping a Host

```bash
ping google.com
```

Tests network connectivity.
