# OSPF (Open Shortest Path First)

## What is OSPF?

OSPF is a link-state routing protocol used inside an organization.

---

## Features

- Fast convergence
- Uses SPF algorithm
- Supports VLSM
- Classless routing

---

## Important Terms

- Router ID
- Area 0
- DR
- BDR
- LSA

---

## Neighbor States

Down

Init

2-Way

ExStart

Exchange

Loading

Full

---

## Useful Commands

Cisco:

```bash
show ip ospf neighbor
show ip ospf interface
show ip route ospf
```

---

## Interview Question

What is Area 0?

Area 0 is the backbone area that connects all other OSPF areas.
