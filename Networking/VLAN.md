# VLAN (Virtual LAN)

## What is a VLAN?

A VLAN logically separates devices into different broadcast domains on the same switch.

---

## Benefits

- Improved security
- Reduced broadcasts
- Better network management

---

## VLAN Types

- Default VLAN
- Data VLAN
- Voice VLAN
- Native VLAN
- Management VLAN

---

## Access Port

Carries one VLAN.

---

## Trunk Port

Carries multiple VLANs using 802.1Q tagging.

---

## Troubleshooting

- Verify VLAN exists
- Check trunk allowed VLANs
- Check native VLAN mismatch
- Verify switchport mode

---

## Interview Question

Difference between access and trunk ports?

Access ports carry one VLAN. Trunk ports carry multiple VLANs.
